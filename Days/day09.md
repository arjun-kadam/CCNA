# IP Addressing 
We will see IPv4 Addressing in this guide.
## IPv4 Address
- It is 32-bit address.
- Total 4 octect(1 octect = 8 bits)
- One octect range from 0-255 Max
- IP Address ---> Network ID + Host ID

==========================================================
