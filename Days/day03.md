## OSI Model
- OSI Means Open System Interconnection.
- OSI is a reference model.
- This is The Image that explains overall OSI Model
<p align="center">             
<img src="Images/OSI_Model.jpg?raw=true" alt="Simplex Mode">

- OSI has 7 layers. Each layer has it's own functions.
- All layers are dependent on each other.

### Which is first layer of OSI Model??
- If you are sender then first layer is Application Layer.
- If you are receiver then first layer is Physical Layer.
